# Aashan ERP Web

Clean Next.js starter for Cloudflare Pages / GitHub deployment.

## Build Settings
Build command: `npm run build`
Output directory: `.next`

## Features
- Dashboard
- Customers
- Jobs / Quotes
- Invoice list
- Manager.io CSV export

## Supabase
This starter UI works without Supabase first. Supabase database can be added in Phase 2.
